# 국방 휴가 체계
![Logo](https://logosbynick.com/wp-content/uploads/2018/03/final-logo-example.png)

## 프로잭트 소개
> 군생활을 하면서 휴가를 한번에 파악하기가 어렵지 않습니까?<br/>
> 휴가를 신청 할 때 복잡한 절차로 인해 귀찮지 않으십니까?<br/>
> 이러한 불편함들은 **국방 휴가 체계**가 해결해 줄 것입니다.
